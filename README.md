## Design Philosophy

The visual identity draws from three aesthetics fused into one:

**1. Classified Documents** вҖ” Redacted text, registration marks, section numbers, diagonal hatching, `[REDACTED]` stamps

**2. Military Command Center** вҖ” Scanlines, grid backgrounds, radar sweeps, status indicators, operational readouts

**3. Retro Terminal** вҖ” JetBrains Mono, neon accents (`#d4f000` lime, `#ff9e9e` coral), noise textures, CRT glow

Every element breathes. Subtle pulse animations, mouse-tracking spotlight effects, and GSAP-powered entrance reveals give the interface a living, operational feel. This isn't a website. It's a command center.
