  <files>src/lib/api/solscan.ts</files>
  <action>
Create src/lib/api/solscan.ts as a server-side-only module with these exports:

1. **Private `solscanHeaders()` function:** Returns `{ token: process.env.SOLSCAN_API_KEY! }`. Import `SOLSCAN_API_BASE` from `@/lib/constants`.

2. **`Transfer` interface:** `{ trans_id: string, block_time: number, time: string, activity_type: string, from_address: string, to_address: string, token_address: string, token_decimals: number, amount: number, flow: 'in' | 'out' }`.

3. **`getAccountTransfers(address, options?): Promise<Transfer[]>`** — GET `/account/transfer`. Options: `flow?: 'in'|'out', token?: string, from?: string, to?: string, page?: number, page_size?: number (default 50), from_time?: number, to_time?: number`. Build URLSearchParams from address + defined options only (skip undefined). Cache: 300s. Return `json.data ?? []`.

4. **`getAllAccountTransfers(address, options?): Promise<Transfer[]>`** — Pagination helper that calls getAccountTransfers in a loop. Fetches page 1, checks if `data.length === page_size`, if so fetches next page. Continues until shorter page or empty result. Returns concatenated array. Max 10 pages safety limit to prevent infinite loops.

5. **`TokenTransfer` interface:** `{ trans_id: string, block_time: number, from_address: string, to_address: string, token_address: string, token_decimals: number, amount: number }`.

6. **`getTokenTransfers(tokenMint, options?): Promise<TokenTransfer[]>`** — GET `/token/transfer`. Options: `from?: string, to?: string, page?: number, page_size?: number (default 50), from_time?: number, to_time?: number`. Build URLSearchParams from address + defined options. Cache: 300s.

7. **`getAllTokenTransfers(tokenMint, options?): Promise<TokenTransfer[]>`** — Same pagination pattern as getAllAccountTransfers but for token transfers.

8. **`Holder` interface:** `{ address: string, amount: number, decimals: number, owner: string, rank: number }`.

9. **`getTokenHolders(tokenMint, page=1, page_size=40): Promise<{ total: number; items: Holder[] }>`** — GET `/token/holders`. CRITICAL: page_size ONLY accepts 10, 20, 30, or 40 (pitfall #3). Type the page_size parameter as `10 | 20 | 30 | 40` to enforce this at compile time. Cache: 300s.

10. **`getTop100Holders(tokenMint): Promise<Holder[]>`** — Helper that calls getTokenHolders for pages 1, 2, 3 (40 items each = 120 max, take first 100). Uses Promise.all for parallel page fetches. Returns sorted by rank, capped at 100.

Export all interfaces and functions as named exports.
